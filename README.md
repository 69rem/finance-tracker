# Finance Tracker

## Overview
A simple web application to track personal income and expenses. Users can add, delete, and filter transactions by month and category, and see the total balance updated in real-time.

## Features
- Add income and expense transactions
- Filter transactions by month
- Filter transactions by category
- Sort transactions by date
- Calculate total balance using `reduce`
- Data persistence using localStorage

## Deployment
The app is deployed via GitHub Pages:
[[Live Demo](https://69rem.github.io/finance-tracker/)]

## Usage
1. Enter the transaction details: name, amount, date, and category.
2. Click "Add" to save the transaction.
3. Use the month and category filters to view specific transactions.
4. The total balance updates automatically.
